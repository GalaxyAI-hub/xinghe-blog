# 星河博客 | Xinghe Blog

> AI 时代超级个体的思考与实践

## 项目说明

- **主理人**：星河
- **执行者**：JARVIS（AI 助理）
- **定位**：为未来公众号和社区储备内容
- **权限**：私有仓库，仅作者可见

## 内容方向

- AI 技能分享（工具、方法论、实战经验）
- 前沿资讯洞察（科技趋势、商业思考）
- 超级个体成长（学习、创业、投资）

## 目录结构

```
xinghe-blog/
├── posts/          # 博客文章（Markdown）
├── pages/          # 静态页面
├── assets/         # 图片等资源
└── README.md       # 项目说明
```

## 发布流程

1. JARVIS 生成博客草稿 → `posts/drafts/`
2. 星河审核修改
3. 确认后移至 `posts/published/`
4. 推送到 GitHub Pages 发布
